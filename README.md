## Деплой на Railway

### 1. Загрузи файлы на GitHub
- Создай новый репозиторий на github.com
- Загрузи все 3 файла: bot.py, requirements.txt, Procfile

### 2. Подключи к Railway
- Зайди на railway.app
- Нажми "New Project" → "Deploy from GitHub repo"
- Выбери свой репозиторий

### 3. Добавь переменные окружения
В Railway → твой проект → Variables добавь:
- BOT_TOKEN = 8923862657:AAHdeOhKabUxww8Yn-x1L3-REJz5yyimdW4
- ADMIN_CHAT_ID = 1211960244

### 4. Deploy!
Railway автоматически установит зависимости и запустит бота.
